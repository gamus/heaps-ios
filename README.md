# Heaps iOS App
Example heaps iOS app

This example heaps ios app will build the [Base2D](https://heaps.io/samples/base2d.html) from heaps and package it for iOS.

## Geting Started
```
git clone https://github.com/thejustinwalsh/heaps-ios.git
cd heaps-ios
sh bootstrap.sh
```
